# Singapore Red Cross Thrift Shop Engagement Initiative Website

This is a vibrant, community-focused website highlighting sustainability, upcycling, and community giving.

## Features
- Clean, modern, eco-friendly design
- Mobile-friendly responsive layout
- Interactive features: QR code generator, donation tally, event countdown
- Modular sections with clear calls-to-action

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- QRCode.js library for QR generation

## Setup
1. Clone or download the project files.
2. Place the following images in the `images/` folder:
   - `red-cross-logo.png`: Red Cross logo
   - `hero-bg.jpg`: Background image of thrift shop items and upcycling workshop
   - `before1.jpg`, `after1.jpg`: Before/after images for tote bag
   - `before2.jpg`, `after2.jpg`: Before/after images for jacket
   - `before3.jpg`, `after3.jpg`: Before/after images for beanie
   - `pre-event.png`, `during.png`, `post-event.png`: Icons for waste management phases
3. Open `index.html` in a web browser to view the site.

## Running Locally
To run a local server (optional):
- If you have Python installed: `python -m http.server 8000`
- Then open `http://localhost:8000` in your browser.

## Customization
- Edit `css/styles.css` for styling changes.
- Modify `js/script.js` for JavaScript functionality.
- Update `index.html` for content changes.

## Color Scheme
